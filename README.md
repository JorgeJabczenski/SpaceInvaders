# SpaceInvaders
Jogo feito inteiramente em C com auxílio da biblioteca N-Curses

### Como rodar
 - Utilize o comando make
 - Para rodar SEM modificações no teclado: ./space
 - Para rodas COM modificações no teclado: ./run.sh
